Sales Data Analytics Dashboard using Python, SQL & Power BI
Project Overview

This project demonstrates a complete end-to-end data analytics workflow for sales data processing, analysis, and business intelligence reporting. The solution transforms raw Excel sales data into meaningful business insights using Python, SQL, and Power BI.

**The project covers:**

Data extraction from Excel files
Data cleaning and preprocessing using Python
SQL database integration
Analytical querying
Interactive Power BI dashboard development
KPI reporting and trend analysis

**Project Objectives**

Automate sales data cleaning and preprocessing
Build a centralized SQL database for reporting
Generate business insights from sales data
Develop interactive dashboards for decision-making
Demonstrate a complete analytics pipeline

**Project Architecture**

Excel Files
     ↓
Python (Pandas Cleaning & Transformation)
     ↓
SQL Database
     ↓
SQL Queries & Analysis
     ↓
Power BI Dashboard
     ↓
Business Insights & KPI Reporting
Data Cleaning Process

The following preprocessing tasks were performed using Python and Pandas:

Removed duplicate records
Handled missing/null values
Standardized column formats
Converted date columns
Cleaned inconsistent values
Prepared structured datasets for SQL storage
